This application is hosted using Google Cloud Platform Cloud Run Service, Cloud Run is googles self managed services for hosting single instance containers, opposed to Kubernetes services that is for managing clusters of containers.|The runtime is a Node.js based Express.js application, a popular JavaScript REST API that can be used to build enterprise grade webb based application.|Express.js can host any type of web application, cross-platform and PWA, it is the de-facto technology when building web applications and almost any Node.js based web application technology is built on top of its API.|CI/CD ready, this application deploys to cloud run as a container on 'git push'.| In a local development environment docker is used to build and test if the app is running in the container as intended.| This service is courtesy of the Google Cloud Platform.
